# Theoretical Reference

Wilkins K, LaFramboise T. *Losing balance: Hardy-Weinberg disequilibrium as a marker for recurrent loss-of-heterozygosity in cancer*. Human Molecular Genetics (2011), 20(24):4831-4839. doi:10.1093/hmg/ddr422.

This repository implements a practical replication-oriented pipeline inspired by the paper.

Scope notes:

- The pipeline focuses on data extraction and genotype matrix construction from GEO SOFT datasets.
- Statistical steps in this repository are operational and exploratory; they are not a full reimplementation of the exact test and MLE framework in the original paper.
